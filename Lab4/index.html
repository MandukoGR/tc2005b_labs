  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    1. Java es un lenguaje orientado a objetos mientras que Javascript es basado en prototipos , esto permite emular la programación orientada a objetos.
    2. Java es un lenguaje tipado y Javascript es debilmente tipado. Las variables en Java estrictamente necesitan un tipo y no pueden modificarse. En Javascript una misma variable puede variar de tipo.
    3. Java es un lenguaje más pesado que Javascript.
    4. Java es un lenguaje de servidor, mientras que Javascript sirve para crear código sólo de cliente, sólo de servidor, o de cliente y servidor.
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    1. Date.now(): Devuelve valor númerico de la hora actual.
    2. Date.parse(): Transforma una cadena que representa una fecha, y devuelve el número de milisegundos desde el 1 de enero de 1970, hora local 00:00:00.
    3. Date.UTC(): Acepta los mismos parámetros que la forma más larga del constructor, y devuelve el número de milisegundos en un objeto Date desde el 1 de enero de 1970, hora universal 00:00:00.
    4. Date.getDay(): Devuelve el día de la semana entre 0 y 6.
    5. Date.getMinutes(): Devuelve los minutos entre 0 y 59.
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    1. sort(): Ordena los elementos del arreglo.
    2. splice(): Añade o elimina elementos de un arreglo.
    3. toString(): Convierte arreglo a string.
    4. unshift(): Añade elementos al inicio del arreglo y devuelve la nueva longitud.
    5. pop(): Elimina el último elemento del arreglo y devuelve su nueva longitud.
  • ¿Cómo se declara una variable con alcance local dentro de una función?

    • Se utiliza la palabra reservada var. Cabe destacar que si se quiere un alcance de bloque (en un ciclo por ejemplo) se pueden usar las palabras reservadas const y let.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • Hacer esto no es la mejor opción ya que puede ocasionar problemas de mantenibilidad. Además si muchas funciones utilizan la variable puede ser mas dificil identificar errores.
